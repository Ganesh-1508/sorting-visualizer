# 📊 Sorting Visualizer

An interactive web application that demonstrates how different sorting algorithms work through real-time animations. This project helps users understand the step-by-step execution of sorting algorithms in a visually engaging way.

## 🚀 Features

* 🔹 Bubble Sort
* 🔹 Selection Sort
* 🔹 Insertion Sort
* 🔹 Merge Sort
* 🔹 Quick Sort
* 🔹 Generate Random Array
* 🔹 Adjustable Animation Speed
* 🔹 Responsive User Interface

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
